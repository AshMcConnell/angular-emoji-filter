AngularJS Emoji Filter

A simple filter for AngularJS to parse text and replace Emoji codes with actual Emoji icons.

Download now (v0.0.2) or install from Bower with bower install angular-emoji-filter

How to use it

            

                

                    
                

                

                    
Installation

                    

                        The filter is available through Bower. Install it with bower install angular-emoji-filter. Alternatively, download the bundle from this page and unzip it.
                    

                    

                    
Basic use

                    

                        Include the JavaScript and CSS files like normal. Make sure you include emoji.min.js after your main AngularJS include.
                    

                    

                        Set up your page as normal. Where you want to display Emoji icons, apply the emoji filter. Note that since the filter will output HTML it must be used in conjuction with the ng-bind-html directive or similar.
                    

                    

                        Add the emoji module as a dependency of any Angular module you need it to be available to.
                    

                    
Customisation

                    

                        If you want to customise the filter, you can fork the repository on GitHub and modify the Gruntfile accordingly. Full details can be found in the README file.
